Fighter Arena

Project Title & Description
Fighter Arena is a 2D fighting game prototype developed during Sprint 1. The objective of this sprint was to implement core gameplay mechanics, focusing on responsive player movement and basic combat actions. The project emphasizes real-time input handling and smooth player control as a foundation for future development.

Features Implemented in Sprint 1
Completed User Stories

Summary of Implemented Features
•	Keyboard-based fighter movement
•	Punch and kick combat mechanics
•	Immediate response to player inputs
•	Basic combat interaction loop

Technologies Used
•	Game Engine: Unity
•	Programming Language: C#
•	Development Environment: Visual Studio / Visual Studio Code

Setup Instructions
Prerequisites
•	Unity Hub installed
•	Compatible Unity Editor version
Installation & Execution
# Clone the repository
git clone https://github.com/org/project.git

# Open Unity Hub
# Click on "Open Project"
# Select the cloned project directory

# Open the main scene and press Play

Controls
•	Move Left / Right: Arrow Keys or A / D
•	Punch: Assigned keyboard key
•	Kick: Assigned keyboard key
